# 최종 테스트 - 크리스마스 프로모션

## 미션 목표
> 12월 이벤트를 위한 개발 요청

## 기능 목록

### 주요 기능
- [ ] 할인 금액 계산
  - [ ] 크리스마스 디데이 할인 : 2023.12.1 ~ 2023.12.25
  - [ ] 평일 할인 : 2023.12.1 ~ 2023.12.31
  - [ ] 주말 할인
  - [ ] 특별 할인
  - [ ] 증정 이벤트
- [ ] 이벤트 배지 부여

### 입력
- [ ] 예상 방문 날짜
- [ ] 주문 메뉴와 개수 입력

### 출력
- [ ] 환영 인사 출력 
- [ ] 식당 예상 방문 날짜 프롬프트
- [ ] 주문 메뉴 프롬프트
- [ ] 이벤트 혜택 미리보기

### 예외

## 사고 흐름
- 메뉴 생성하기 : Menu, Menus
- 주문하기 : Order, Orders
- 이벤트 적용하기 : Event, Events
- 계산하기 : Payment
